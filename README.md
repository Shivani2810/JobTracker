# 🚀 Job Tracker (MERN Stack)

A full-stack Job Tracker application to manage job applications efficiently with authentication and status tracking.

---

## 🌐 Live Demo

👉 https://job-tracker-ecru-ten.vercel.app

---

## 📌 Features

* 🔐 User Authentication (JWT)
* ➕ Add Job Applications
* ✏️ Edit Jobs
* ❌ Delete Jobs
* 📊 Track Job Status (Applied, Interview, Offer, Rejected)
* 📋 Dashboard View

---

## 🛠️ Tech Stack

**Frontend:**

* React (Vite)
* Axios
* CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB (Atlas)

**Deployment:**

* Frontend: Vercel
* Backend: Render

## ⚙️ Installation & Setup

### 1. Clone the repo

```bash
git clone https://github.com/Shivani2810/JobTracker.git
cd JobTracker
```

---

### 2. Setup Backend

```bash
cd backend
npm install
```

Create `.env` file inside backend:

```env
PORT=8000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

Run backend:

```bash
npm run dev
```

---

### 3. Setup Frontend

```bash
cd frontend
npm install
```

Create `.env` file inside frontend:

```env
VITE_API_BASE_URL=http://localhost:8000/api
```

Run frontend:

```bash
npm run dev
```
